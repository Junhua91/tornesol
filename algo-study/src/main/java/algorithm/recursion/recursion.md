####1. master 公式
```
 T [n] = aT[n/b] + O(N^d)
 ①当d<logb a时，时间复杂度为O(n^(logb a))
 ②当d=logb a时，时间复杂度为O((n^d)*logn)
 ③当d>logb a时，时间复杂度为O(n^d)
 n/b => 子过程的样本量；a=>子过程发生的次数；O(N^d): 子过程时间复杂度
 该公式的适用范围：子过程的规模一致
```


####2. 时间复杂度分析
1. 二分
```
T(N) = 2*T(n/2) + O(1) => a=2,b=2, d=0 => d<log(b,a) = > 
O(n^(log(b,a))) = O(n^log(2,2)) = O(n)
```
